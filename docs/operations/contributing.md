title: Contributing
---

We welcome pull requests and bug reports.

1. Open an issue describing the change.
2. Run `cargo fmt`, `cargo clippy --all-targets`, and `cargo test --workspace`.
3. For end-to-end modifications, also run `bash tools/check/check.sh` and attach the summarized results.
4. Submit the PR against `main` with a short summary of the generated plans or benchmarks.
