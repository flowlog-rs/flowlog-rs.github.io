  <StyledFlowLog /> workflows have two simple steps:
</div>

1. Write a Datalog program.
2. Run it by providing inputs — either as files in batch mode or as incremental updates in an interactive session.

In this section, we’ll start with a small example program and a tiny dataset so you can see what <StyledFlowLog /> programs look like and how to execute them. Then we’ll repeat the same program in incremental mode and update its inputs over time to see results change immediately.
